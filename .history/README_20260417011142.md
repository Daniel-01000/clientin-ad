# ClientIn · Ad Maker

A premium internal content engine for generating social media posts, ad copy, and marketing content from ClientIn app screenshots — powered by OpenAI GPT-4o.

## What it does

- **Upload** multiple ClientIn app screenshots (drag & drop)
- **Analyze** each screenshot with GPT-4o Vision to identify features and flows
- **Generate** multiple pieces of sharp, startup-quality content including:
  - Instagram captions (short + long)
  - Carousel ideas
  - Ad headlines & body copy
  - LinkedIn founder posts
  - Short-form video concepts
  - Hook variations
  - CTA sets
  - Premium promo concepts
- **Filter** by content type, industry, and tone
- **Save** favorites and copy to clipboard instantly

## Industries supported

Barbershop · Café · Salon · Restaurant · General

## Tone options

Premium · Founder-Led · Direct · Educational · Bold

## Tech stack

- **Next.js 15** App Router
- **TypeScript**
- **Tailwind CSS v4**
- **OpenAI GPT-4o** (vision + chat)
- `react-dropzone` · `lucide-react` · `clsx` · `uuid`

## Getting started

1. Copy `.env.local.example` to `.env.local`:
   ```bash
   cp .env.local.example .env.local
   ```
2. Add your OpenAI API key to `.env.local`
3. Run the dev server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)
